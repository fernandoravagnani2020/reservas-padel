# 🎾 NEGRO PADEL - Sistema de Reservas

Sistema completo de gestión de reservas para cancha de pádel, conectado a Google Sheets.

## 🌐 Links de Acceso

- **Vista Pública (Clientes):** [Ver Turnos Disponibles](./index.html)
- **Panel Admin:** [Administrar Reservas](./admin.html)

## ✨ Características

- ✅ **Semana Móvil** - Se actualiza automáticamente cada día
- ✅ **Turnos Fijos** - Se repiten todas las semanas
- ✅ **Sincronización en Tiempo Real** - Conectado a Google Sheets
- ✅ **WhatsApp Integrado** - Click directo para reservar
- ✅ **Responsive** - Funciona en celular, tablet y PC

## 🔧 Tecnologías

- HTML5 / CSS3 / JavaScript
- Google Apps Script (API)
- Google Sheets (Base de datos)

## 📱 Uso

### Para Clientes:
1. Entrá a la vista pública
2. Mirá los turnos disponibles (verde) y ocupados (rojo)
3. Click en un turno disponible para reservar por WhatsApp

### Para Administradores:
1. Accedé al panel admin (link privado)
2. Agregá/eliminá reservas
3. Marcá turnos como fijos
4. Todo se sincroniza automáticamente con Google Sheets

## 📊 Google Sheets

El sistema usa 3 hojas:
- **RESERVAS** - Vista semanal actualizada automáticamente
- **TURNOS_FIJOS** - Turnos que se repiten cada semana
- **HISTORIAL** - Registro completo de todas las reservas

## 🔐 Seguridad

- El panel de administración debe mantenerse privado
- Solo compartir el link público con clientes
- Los datos se guardan en tu Google Sheet personal
